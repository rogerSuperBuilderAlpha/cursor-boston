# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial open source release
- Community platform with events, talks, and blog
- User authentication with Firebase (Email, Google, GitHub)
- Member directory with social profiles
- Talk submission system
- Event request system
- Discord and GitHub OAuth integration
- Responsive design with dark theme
- Accessibility features (WCAG compliant)

## [0.1.0] - 2026-01-27

### Added
- Initial release of Cursor Boston community platform
- Next.js 14 App Router implementation
- Firebase Authentication integration
- Firestore database for user profiles and submissions
- Blog system with markdown support
- Events listing and management
- Talks archive and submission form
- Member directory with social links
- User profile management
- Email notifications for submissions (via Firebase Trigger Email extension)
- GitHub webhook integration for PR tracking
- Discord server membership verification
- Responsive navigation with mobile menu
- Footer with community links
- About page
- Security rules for Firestore
- Environment variable configuration
- Comprehensive documentation

### Security
- Firestore security rules implemented
- XSS prevention in email templates
- Secure OAuth callback handling
- Environment variable validation
